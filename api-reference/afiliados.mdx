---
title: "Validação de Afiliados"
api: "GET /affiliate/validate/{code}"
description: "Verifique a validade de códigos de afiliados para aplicar descontos ou comissões."
---

Este endpoint permite que você valide se um código de afiliado existe e está ativo antes de processar uma venda ou cadastro em seu sistema.

### Path Parameters

<ParamField path="code" type="string" required>
  O código único do afiliado (ex: `PROMO2024`).
</ParamField>

### Resposta de Sucesso

<ResponseField name="success" type="boolean">
  Indica se a requisição foi processada.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Propriedades do Objeto">
    <ResponseField name="valid" type="boolean">
      `true` se o código for válido e estiver ativo.
    </ResponseField>
    <ResponseField name="code" type="string">
      O código validado.
    </ResponseField>
    <ResponseField name="discount" type="number">
      Valor ou porcentagem de desconto associado (se houver).
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.zynxwallet.com.br/api/v1/affiliate/validate/PROMO2024" \
  -H "X-API-Key: zynx_sua_chave_aqui"
```
</RequestExample>

<ResponseExample>
```json Resposta
{
  "success": true,
  "data": {
    "valid": true,
    "code": "PROMO2024",
    "discount": 10.0
  }
}
```
</ResponseExample>

<Tip>
  Use este endpoint no momento do checkout para validar cupons de desconto em tempo real.
</Tip>
