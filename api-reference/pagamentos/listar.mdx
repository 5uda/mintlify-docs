---
title: "Listar Pagamentos"
api: "GET /payment/list"
description: "Retorna uma lista paginada de todos os seus pagamentos."
---

### Query Parameters

<ParamField query="status" type="string">
  Filtrar por status (`PENDING`, `COMPLETED`, `FAILED`).
</ParamField>

<ParamField query="limit" type="number" default="50">
  Quantidade por página (máx 1000).
</ParamField>

<ParamField query="offset" type="number" default="0">
  Pular registros.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.zynxwallet.com.br/api/v1/payment/list?status=COMPLETED&limit=10' \
  -H 'X-API-Key: zynx_sua_chave_aqui'
```
</RequestExample>

<ResponseExample>
```json Resposta
{
  "success": true,
  "data": [
    {
      "id": "pay_abc123",
      "status": "COMPLETED",
      "value": 15300,
      "netValue": 15000,
      "updatedAt": "2024-02-16T15:05:00Z"
    }
  ]
}
```
</ResponseExample>
