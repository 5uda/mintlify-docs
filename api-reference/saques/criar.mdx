---
title: "Solicitar Saque PIX"
api: "POST /withdraw/create"
description: "Transfere saldo da sua conta Zynx para uma chave PIX externa."
---

### Parâmetros do Corpo (JSON)

<ParamField body="amount" type="number" required>
  Valor em reais.
</ParamField>

<ParamField body="pixKey" type="string" required>
  Chave PIX de destino.
</ParamField>

<ParamField body="pixKeyType" type="string" required>
  `CPF`, `CNPJ`, `EMAIL`, `PHONE` ou `RANDOM`.
</ParamField>

<ParamField body="description" type="string">
  Descrição da retirada.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST 'https://api.zynxwallet.com.br/api/v1/withdraw/create' \
  -H 'X-API-Key: zynx_sua_chave_aqui' \
  -H 'Content-Type: application/json' \
  -d '{
    "amount": 500.00,
    "pixKey": "12345678900",
    "pixKeyType": "CPF",
    "description": "Retirada de Lucro"
  }'
```
</RequestExample>

<ResponseExample>
```json Resposta
{
  "success": true,
  "message": "Saque solicitado com sucesso"
}
```
</ResponseExample>
