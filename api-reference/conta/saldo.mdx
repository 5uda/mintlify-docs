---
title: "Consultar Saldo e Estatísticas"
api: "GET /user/balance"
description: "Retorna o saldo disponível, valores pendentes e estatísticas de uso."
---

### Resposta de Sucesso (200 OK)

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Propriedades do Objeto">
    <ResponseField name="balance" type="object">
      <Expandable title="Detalhes do Saldo">
        <ResponseField name="total" type="number">
          Saldo total em centavos.
        </ResponseField>
        <ResponseField name="pendingPayments" type="number">
          Pagamentos pendentes em centavos.
        </ResponseField>
        <ResponseField name="totalWithPending" type="number">
          Saldo total incluindo pendentes em centavos.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="statistics" type="object">
      <Expandable title="Estatísticas">
        <ResponseField name="totalReceived" type="number">
          Total recebido em centavos.
        </ResponseField>
        <ResponseField name="completedPayments" type="number">
          Número de pagamentos concluídos.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.zynxwallet.com.br/api/v1/user/balance' \
  -H 'X-API-Key: zynx_sua_chave_aqui'
```
</RequestExample>

<ResponseExample>
```json Resposta
{
  "success": true,
  "data": {
    "balance": {
      "total": 1250050,
      "pendingPayments": 45000,
      "totalWithPending": 1295050
    },
    "statistics": {
      "totalReceived": 5000000,
      "completedPayments": 342
    }
  }
}
```
</ResponseExample>

<Note>
  **Centavos:** Todos os valores monetários retornados nos campos de saldo estão em **centavos** (inteiros).
</Note>
