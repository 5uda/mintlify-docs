---
title: "Transferência Interna"
api: "POST /api/account/transfer"
description: "Transfere saldo instantaneamente para outro usuário Zynx."
---

### Request Body

<ParamField body="amount" type="number" required>
  Valor da transferência.
</ParamField>

<ParamField body="toUserEmail" type="string" required>
  E-mail do destinatário Zynx.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.zynxwallet.com.br/api/account/transfer" \
  -H "X-API-Key: zynx_sua_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 10.00,
    "toUserEmail": "destino@exemplo.com"
  }'
```
</RequestExample>

<ResponseExample>
```json Resposta
{
  "success": true,
  "message": "Transferência realizada com sucesso"
}
```
</ResponseExample>
