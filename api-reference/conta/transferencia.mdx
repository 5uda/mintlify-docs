---
title: "Enviar Dinheiro (Reembolso/Transferência)"
api: "POST /payment/send"
description: "Realiza um envio direto de valores. Ideal para automação de reembolsos."
---

### Parâmetros do Corpo (JSON)

<ParamField body="amount" type="number" required>
  Valor em reais.
</ParamField>

<ParamField body="pixKey" type="string" required>
  Chave PIX de destino.
</ParamField>

<ParamField body="pixKeyType" type="string" required>
  `CPF`, `CNPJ`, `EMAIL`, `PHONE` ou `RANDOM`.
</ParamField>

<ParamField body="description" type="string">
  Descrição da transferência.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST 'https://api.zynxwallet.com.br/api/v1/payment/send' \
  -H 'X-API-Key: zynx_sua_chave_aqui' \
  -H 'Content-Type: application/json' \
  -d '{
    "amount": 25.50,
    "pixKey": "cliente@email.com",
    "pixKeyType": "EMAIL",
    "description": "Reembolso Pedido #99"
  }'
```
</RequestExample>

<ResponseExample>
```json Resposta
{
  "success": true,
  "message": "Transferência realizada com sucesso"
}
```
</ResponseExample>
